# Closeout

## Objective

Represent `ABYSS / STCW` publicly without exposing the active operational tree.

## Final intended model

- public technical brief for architecture and scope
- private cleaned repository for controlled technical review

## Public posture

This project should be treated publicly as:

- a technical brief
- a due-diligence layer
- a portfolio artifact

It should not be treated publicly as:

- an open-source release
- a public demo dataset
- a repo that mirrors the live operational tree

## Rule before moving on

Do not publish the raw `CRM-STCW` directory directly.
